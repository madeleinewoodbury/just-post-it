# Just-Post-It

This is a CRUD application built with node.js and bootstrap. Users can register in order to use the application. The purpose of the app is to share posts with other users.

## Demo

To view the page [click here](http://www.just-post-it.online/)

## Dependencies

- [Font Awesome](https://fontawesome.com/?from=io)
- [Bootstrap](https://getbootstrap.com/)
- [Cloudinary](https://cloudinary.com/)
